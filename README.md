# CREDIXA ATM Simulator

Java Swing and SQLite based ATM Simulator.

## Features

- Login System
- Deposit Money
- Withdraw Money
- Change PIN
- Mini Statement
- SQLite Database Connectivity

## Technologies Used

- Java
- Swing
- SQLite
- JDBC
